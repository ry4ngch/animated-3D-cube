# 3D Cube Viewer

A lightweight web application that displays an interactive 3D cube with animation.  
The UI allows you to adjust cube size, rotation, animation speed, toggle exploded view, and control the visibility of cube faces.

## Features

- **3D Cube Rendering**  
  Displays a pure CSS-based 3D cube with smooth animation.

- **Adjustable Cube Size**  
  Change the cube's dimensions through sliders or input fields.

- **Rotation Controls**  
  Modify rotation on the X, Y, and Z axes in real time.

- **Animation Toggle & Animation Type**  
  Turn animation on/off and adjust animation type.

- **Exploded View**  
  Expand the cube so each face moves outward for structural viewing.

- **Visible Plane Control**  
  Toggle cube central faces ON/OFF.

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/ry4ngch/animated-3D-cube.git
cd animated-3D-cube
```

### 2. Open the App
You can either
- Open `index.html` directly in your browser, **or**
- Start a lightweight local server:
```bash
npx serve
```

## Project Structure

```bash
/animated-3D-cube
│── index.html        # Main UI and cube container and logic
└── style.css         # Cube styling, animation 
```

## Usage

1. Open the page in your web browser.

2. Use the UI controls to:
- Change cube size
- Adjust rotation angles
- Enable/disable animations
- Modify animation type
- Toggle exploded view
- Show/hide individual cube faces

3. The cube preview updates instantly based on your settings.
